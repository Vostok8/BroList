# BroList

Репозиторий со списком доменов и IP/CIDR.

## Как это работает

- Исходный файл: `brolist.txt`.
- Скрипт `scripts/resolve.py` читает файл, отделяет домены от статических IPv4/IPv6/CIDR и резолвит домены в IPv4 + IPv6.
- Для временно нерезолвящихся доменов используется последний успешный результат до 24 часов.
- Выходные файлы обновляются автоматически через GitHub Actions каждый час.

## Формат `brolist.txt`

- Пустые строки и строки, начинающиеся с `#`, игнорируются.
- Домены (например `openai.com`) резолвятся в IPv4 и IPv6.
- IPv4/IPv6 и IPv4/IPv6 CIDR (например `1.2.3.4`, `91.108.56.0/22` или `2001:b28:f23d::/48`) попадают в итоговые списки как есть.
- Дубликаты автоматически удаляются.

## Выходные файлы

- `ips.txt` — универсальный список IPv4/IPv6/CIDR, по одному значению на строку.
- `ips_v4.txt` — список только IPv4/CIDR.
- `ips_v6.txt` — список только IPv6/CIDR.
- `wireguard_allowed_ips.txt` — готовая строка `AllowedIPs = ...`.
- `shadowsocks_ips.txt` — список IPv4/CIDR для ручного использования.
- `amnezia_sites.json` — список доменов в формате JSON (`[{ "hostname": "example.com" }]`).
- `state/resolve_state.json` — состояние последнего успешного резолва для политики 24 часов.

## Локальный запуск

```bash
python scripts/resolve.py
```
