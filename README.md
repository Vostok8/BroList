# BroList

Репозиторий со списком доменов и IP/CIDR для обхода блокировок.

## Как это работает

- Исходный файл: `brolist.txt`.
- Скрипт `scripts/resolve.py` читает файл, отделяет домены от статических IPv4/CIDR и резолвит домены в IPv4.
- Для временно нерезолвящихся доменов используется последний успешный результат до 24 часов.
- Выходные файлы обновляются автоматически через GitHub Actions каждый час.

## Формат `brolist.txt`

- Пустые строки и строки, начинающиеся с `#`, игнорируются.
- Домены (например `openai.com`) резолвятся в IPv4.
- IPv4 и IPv4 CIDR (например `1.2.3.4` или `91.108.56.0/22`) попадают в итоговые списки как есть.
- IPv6 строки в текущем режиме не экспортируются (режим `IPv4 only`).

## Выходные файлы

- `ips.txt` — универсальный список IPv4/CIDR, по одному значению на строку.
- `ips_v4.txt` — тот же список IPv4/CIDR.
- `ips_v6.txt` — служебный файл (пустой в режиме `IPv4 only`).
- `wireguard_allowed_ips.txt` — готовая строка `AllowedIPs = ...`.
- `shadowsocks_ips.txt` — список IPv4/CIDR для ручного использования.
- `state/resolve_state.json` — состояние последнего успешного резолва для политики 24 часов.

## Локальный запуск

```bash
python scripts/resolve.py
```
